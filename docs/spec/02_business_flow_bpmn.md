# 業務フロー図（BPMN風・テキスト）

## Swimlane
- Lane A: 依頼元
- Lane B: 受託オペレーション
- Lane C: 医師
- Lane D: 経理

## Main Flow
1. A: 案件作成 + 画像アップロード + 送信
2. B: 受付チェック
3. B: Gateway(不備?)
   - Yes -> B: 不備差戻し -> A: 再提出 -> B: 受付チェックへ戻る
   - No  -> B: 受付完了
4. B: 医師割当
5. C: 読影実施 + 提出
6. B: QC確認
7. B: Gateway(QC差戻し?)
   - Yes -> C: 修正読影 -> B: QC確認へ戻る
   - No  -> B: 納品確定
8. A: 納品結果参照/PDF取得
9. D: 月次請求生成 + 月次医師支払生成

## SLA/Event
- タイマーイベント: 受付待ち24h超でアラート
- タイマーイベント: 割当後48h超で医師リマインド
- タイマーイベント: QC待ち24h超でアラート
